#ch09 mysql


mysql是一套免費的資料庫

WAMP或者LAMP中的M代表的就是mysql

雖然是免費的資料庫不過該有的功能也都有

範例將分為基本的連線及查詢

要注意的一點是範例所使用的mysql方法將在php7中被移除

須改用mysqli或者pdo

mysqli的使用方法https://github.com/kuromikan/php_mysqli

pdo的使用方法https://github.com/kuromikan/php_pdo
