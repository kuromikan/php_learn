#ch06 資料傳遞


可以透過網頁的輸入框、checkbox、select等等把值帶向其他網頁

或者是在網址後面帶參數

收到這些資料的網頁可以拿來做另外的運算

比較常用的像是登入、資料輸入、多重選項都可以這樣傳遞
