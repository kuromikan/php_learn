#ch04 陣列


陣列是一個可以儲存很多值的型態

他不會像一般變數只能同時間儲存一個值

陣列可以在不同空間放入不同的值

以實體的概念來看一個一為陣列就像是一排的格子可以讓你放東西

二維陣列則是一個平面切成很多格子放東西

而三維陣列就是一個立方體
