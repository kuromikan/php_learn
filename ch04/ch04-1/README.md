#ch04-1 一維陣列


一維陣列的概念就像是建立一個一排的儲存空間

	□  □  □  □  □  

然後可以在這些空間分別放入想放入的值

	□  □  □  □  □  
	a  b  c  d  e
	
陣列在沒有指定給他哪個位置開始放值的話都是從0開始

	□  □  □  □  □  
	0  1  2  3  4
