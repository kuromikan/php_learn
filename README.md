#ch07 ajax


ajax可以用來解決傳送資料時造成頁面刷新的狀況

ajax是送出一個資料請求，然後伺服器回傳資料給他

不會造成頁面刷新時的閃爍，在接收資料上由於不是傳整張完整的網頁速度也較快
